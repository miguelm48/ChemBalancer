Chemical Equation Balancer

How to Use

Type a chemical equation in the box above, using this format:

  • Use spaces around the “+” and “→” symbols.
  • Write reactants and products like this: H2 + O2 -> H2O
  • Use correct capitalization (e.g., NaCl, not nacl).
  • Make sure to include both sides of the equation with an arrow.
  • Press Balance to see the result!
